# Product App

**Bulk products upload with multiple images**

## Features

- Create product list to send to API 
- Add new product to list with multiple images
- Wait for uploading progress
- View products with images

## Tech Stack
- React
- Next.js
- TailwindCSS

## Environment Variables

To run this project, you will need to add the following environment variables to your .env.development file

`NEXT_PUBLIC_API_BASE` *example: http://localhost:8080*

`NEXT_PUBLIC_RESOURCE_PATH` **To access product images, where images are stored from backend** 
*example:/home/{user}/product-data or /desktop/product-data*

## Run Project
Go inside the project and run with npm
```bash
  npm install
  npm run dev
```
Go to browser and access pages: http://localhost:3000
